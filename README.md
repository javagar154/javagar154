A full-stack fintech platform designed to improve financial literacy through interactive learning and simulated investing.

#### Key Features:
- 📚 Developed learning modules covering budgeting, savings, SIP planning, and investment concepts.
- 📈 Built a stock trading simulator that enables users to practice trading with virtual money.
- 💰 Implemented a budget tracker to help users monitor expenses effectively.
- 🧮 Integrated an SIP calculator for long-term investment planning.
- 🔐 Implemented secure authentication and role-based authorization using Spring Security.
- 🔄 Developed RESTful APIs for seamless frontend-backend communication.
- 🏗️ Followed layered architecture (Controller → Service → Repository) to improve scalability and maintainability.

A cloud-native attendance management system that automates attendance tracking using facial recognition technology and AWS services.

- 🗄️ Stored attendance records securely using Amazon RDS.

- 🔔 Configured Amazon SNS for real-time notifications and alerts.
- 🚀 Improved operational efficiency by minimizing manual attendance processes.
